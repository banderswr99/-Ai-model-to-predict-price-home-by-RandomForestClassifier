House Price Prediction

A simple Machine Learning project that predicts house prices using Random Forest and Scikit-learn.

Features

* Load housing data from CSV
* Train a Random Forest model
* Predict house prices
* Evaluate model performance using Mean Absolute Error (MAE)

Technologies

* Python
* Pandas
* Scikit-learn
* Jupyter Notebook

Dataset Columns

* space
* rooms
* bathrooms
* age
* district_code
* price

Installation

pip install pandas scikit-learn

Run

python first.py

Example Prediction

Input:

[350, 5, 4, 3, 104]

Output:

1100000
